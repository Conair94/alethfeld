Let a triple $(a, b, c)$ of positive real numbers satisfy the equations
$$2(a^2+1) = 3(b^2+1) = 4(c^2+1) \quad \text{and} \quad ab + bc + ca = 1$$

**Claimed**: Show that $a + b + c > 2$.

## Result

**THE THEOREM IS FALSE.**

A counterexample exists: the unique solution satisfying both constraints has $a + b + c < 2$.

## Proof of Falsity

### Step 1: Parametrization

Let $k = 2(a^2+1) = 3(b^2+1) = 4(c^2+1)$ be the common value.

From the constraints:
- $a^2 = \frac{k-2}{2}$, so $a = \sqrt{\frac{k-2}{2}}$
- $b^2 = \frac{k-3}{3}$, so $b = \sqrt{\frac{k-3}{3}}$
- $c^2 = \frac{k-4}{4}$, so $c = \sqrt{\frac{k-4}{4}}$

For $a, b, c$ to be positive real numbers, we require $k > 4$.

### Step 2: The Constraint Function

Define $f(k) = ab + bc + ca$ for $k > 4$.

This function is:
- Continuous on $(4, \infty)$
- Strictly increasing (each of $a, b, c$ increases with $k$)
- $\lim_{k \to 4^+} f(k) = \sqrt{1/3} \approx 0.577$
- $\lim_{k \to \infty} f(k) = \infty$

By the Intermediate Value Theorem, there exists a unique $k^* > 4$ such that $f(k^*) = 1$.

### Step 3: Numerical Computation

Solving $f(k) = 1$ numerically:

$$k^* \approx 4.1739130435$$

- $a = \sqrt{\frac{k^*-2}{2}} \approx 1.0426$
- $b = \sqrt{\frac{k^*-3}{3}} \approx 0.6255$
- $c = \sqrt{\frac{k^*-4}{4}} \approx 0.2085$

### Step 4: The Counterexample

$$a + b + c \approx 1.0426 + 0.6255 + 0.2085 = 1.8766 < 2$$
